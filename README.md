# 🛍️ Customer Behavior Analysis | Data Analytics Project

##  Overview

This project demonstrates an end-to-end Data Analytics workflow, starting from raw data processing to business intelligence dashboard creation. The objective is to analyze customer purchasing behavior, uncover meaningful insights, and present them through interactive visualizations and reports.

The project follows a complete analytics pipeline including data cleaning, exploratory data analysis (EDA), SQL-based analysis, dashboard development, and business reporting.

---

##  Dataset

The dataset contains customer purchasing information, including demographic details, shopping preferences, purchase history, payment methods, product categories, ratings, and other customer behavior attributes.

### Key Features

- Customer ID
- Age
- Gender
- Category
- Purchase Amount
- Previous Purchases
- Review Rating
- Payment Method
- Shipping Type
- Discount Applied
- Location
- Season
- Purchase Frequency
- Item Purchased

##  How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/Customer-Behavior-Analysis.git
```

### 2. Install Required Python Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### 3. Run the Jupyter Notebook

Open:

```
Customer_Behavior_EDA.ipynb
```

and execute all cells.

### 4. PostgreSQL

- Create a PostgreSQL database.
- Import the cleaned dataset.
- Execute the SQL queries provided in the `SQL` folder.

### 5. Power BI

Open:

```
Customer_Behavior_Dashboard.pbix
```

Refresh the data source if required.
